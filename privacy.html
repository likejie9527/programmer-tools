隐私政策

最后更新：2025年12月

1. 数据收集说明

本工具为纯前端运行工具，所有计算、转换、编辑均在用户本地浏览器完成， 我们不会上传、存储或收集任何用户输入的内容、文件或数据。

2. 广告服务

本站使用第三方广告服务（Google AdSense）维持运营。 广告商可能使用 Cookie、像素标签等技术收集访问信息，用于展示个性化广告。

3. Cookie 使用

本站使用 Cookie 保存用户偏好设置（如主题、历史记录）， 仅用于提升使用体验，不用于追踪或收集个人信息。

4. 第三方服务

本站可能使用以下第三方服务：

  • Google AdSense（广告展示）
  • Cloudflare / Vercel / GitHub Pages（网站托管）

5. 联系方式

如对隐私政策有疑问，可通过页面留言联系我们。
